ClilentApp main() creates Client, Client run() requests server connection, ServerApp main() creates Server, Server run() accepts connections and creates Connection, Connection run() asks the it.polimi.ingsw.clientGraphic starting info, than launch Server.lobby(Connection c) which manages the list of connections and creates a RemoteView for each it.polimi.ingsw.clientGraphic. RemoteView observes the model and then using Connection sends the updated model to Client with serialization.
